## 프로젝트 소개
Next.js와 TypeScript를 사용하여 PokeAPI를 기반으로 제작한 포켓몬 도감 프로젝트입니다.

## 주요 기능

### 1. 좋아요 및 북마크 기능
- MongoDB와 Prisma를 활용하여 좋아요 및 북마크 기능 구현.

### 2. 로그인 및 로그아웃
- Auth0를 사용한 로그인, 로그아웃 기능

### 3. 상태 관리
- Zustand를 활용한 상태 관리.
